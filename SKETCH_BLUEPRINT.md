# Sketch blueprint — copy this onto paper BY HAND (digital sketch = 0 marks)

Goal for full marks: **4+ clear sections, headings + text + figures, varied chart
types, clean and tidy.** Draw on plain/grid paper, photograph or scan to a single
PDF, commit it to the repo, and link the GitHub PDF URL on Moodle.

Lay it out like the real page (one tall scrolling column). Label each box with its
chart type so the marker sees variety. You don't need every one of the 17 charts —
pick ~10–12 representative ones across all five sections.

Tips for the 2% / no-penalty:
- Write real section headings and a sentence of dummy text under each (shows
  storytelling intent, not just boxes).
- Make the chart shapes recognisably different (map vs bars vs line vs scatter vs
  heatmap vs dumbbell) — "detailed, creative and varied charts and maps".
- Keep it tidy: straight-ish boxes, legible labels, one column.
- Put your name + date somewhere on the sheet.
